  <h2 align="center">Golang Web App Scaffolding</h2>
  <p align="center">
    This project aims to be a lightweight and simple web app framework that handles authentication, sessions, routing, errors and requests for small web apps. The intention is that this framework can be importing into an existing headless Go application to provide a web based GUI for management/interaction with said service. However I am also using it for small web apps that I am developing. It is currently in a pre-alpha stage and is not sutable for use.
  </p>
